# Harvard CS50's Introduction to Programming with Python (CS50P)

This repository contains my solutions to the problem sets from **Harvard University's CS50P** course.

## 📂 Repository Structure
The solutions are organized by weeks, covering topics from basic variables to object-oriented programming.

- **Week 0:** Functions, Variables
- **Week 1:** Conditionals
- **Week 2:** Loops
- **Week 3:** Exceptions
- **Week 4:** Libraries
- **Week 5:** Unit Tests
- **Week 6:** File I/O
- **Week 7:** Regular Expressions
- **Week 8:** Object-Oriented Programming (OOP)
- **Week 9:** Et Cetera

## 🚀 Technologies Used
- **Language:** Python 3
- **Tools:** VS Code, Git, GitHub
